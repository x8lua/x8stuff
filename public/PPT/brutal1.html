原體
Elementary Body / EB
堅硬的「種子」— 傳播者 負責在外面跑來跑去傳染人。在細胞外就像一塊沒生命的石頭 — 堅硬、休眠、極具感染力。

早期科學家甚至分不清它到底是生物還是無生物，因為在細胞外它毫無生命跡象。
🏭
始體
Reticulate Body / RB
工廠模式 — 瘋狂生產 躲在細胞裡瘋狂生小孩。一進入宿主細胞就立刻啟動，變成瘋狂的生產線，大量複製自己。

它既像病毒又像細菌的特性，讓早期科學家非常頭大 — 必須寄生在別人的細胞裡才能活。

治療方法
TREATMENT PROTOCOL
早期發現
完全可以根治
CRITICAL // 衣原體感染完全可治癒 — 抗生素療程完成後幾乎 100% 清除病菌
PROTOCOL A // SINGLE-DOSE
單次療程
Azithromycin 阿奇黴素
  • 1g 單次口服，一次解決
  • 依從性高，不易漏服
  • 適合擔心忘記服藥者
  • 費用相對較高
PROTOCOL B // 7-DAY COURSE
七日療程
Doxycycline 多西環素
  • 100mg 每日兩次，連服 7 天
  • 療效更穩定，清除率更高
  • 孕婦使用 Amoxicillin 替代
  • 必須嚴格按時服藥
WARNING // 伴侶必須同時接受治療 — 否則出現乒乓效應 (Ping-Pong Effect)，反覆交叉感染
